## D1 вЂ” Migration `supabase/migrations/<ts>_stock_consumption.sql`

```
stk_consumption (
  id uuid pk default gen_random_uuid(),
  org_id text not null,
  fin_product_id uuid not null,        -- the sold service/product (fin_products)
  item_id uuid not null,               -- stk_items consumed
  qty_per_unit numeric not null,       -- consumed per 1 unit sold/performed
  note text,
  created_at/updated_at timestamptz default now(),
  unique(org_id, fin_product_id, item_id)
)
```
org_guc RLS + `app_ledger` grants exactly like `20260702130000_stock.sql`. Orchestrator applies to prod; agent writes file + Drizzle table in `pg-schema/stock.ts`.

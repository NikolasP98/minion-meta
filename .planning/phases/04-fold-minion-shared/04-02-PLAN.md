    - minion-shared/src/utils/index.ts — to replace with shim
  </read_first>
  <action>
    Transform `minion-shared/` into a deprecation shim pointing at `@minion-stack/shared`.

    1. Update `minion-shared/package.json`:
       - Bump `"version"` from `"0.1.0"` to `"0.2.0"`
       - Add `@minion-stack/shared` as a peer dependency (the shim re-exports from it):
         ```json
         "peerDependencies": {
           "@minion-stack/shared": "^0.1.0"
         }
         ```
       - Add a deprecation note in the `"description"` field:
         `"DEPRECATED: Use @minion-stack/shared instead. This package re-exports from @minion-stack/shared for backward compatibility."`

    2. Replace `minion-shared/src/index.ts` with the shim:
       ```typescript
       // DEPRECATED: This package is deprecated. Use @minion-stack/shared instead.
       // https://www.npmjs.com/package/@minion-stack/shared
       console.warn(
         '[minion-shared] This package is deprecated. ' +
         'Please migrate to @minion-stack/shared: ' +
         'https://www.npmjs.com/package/@minion-stack/shared'
       );
       export * from '@minion-stack/shared';
       ```

    3. Replace `minion-shared/src/gateway/index.ts` with:
       ```typescript
       // DEPRECATED: Use @minion-stack/shared/gateway instead.
       export * from '@minion-stack/shared/gateway';
       ```

    4. Replace `minion-shared/src/utils/index.ts` with:
       ```typescript
       // DEPRECATED: Use @minion-stack/shared/utils instead.
       export * from '@minion-stack/shared/utils';
       ```

    5. Build the shim: `cd /home/nikolas/Documents/CODE/AI/minion-shared && npm run build`
       (This runs `tsc`. The shim imports from `@minion-stack/shared` — tsc needs the types. Since `@minion-stack/shared` is a peerDep and not installed in minion-shared/, tsc may fail on the import. If it does, add `"skipLibCheck": true` is already set, but the import itself needs the type definitions. Work around: temporarily install @minion-stack/shared into minion-shared's devDependencies for the build: `npm install --save-dev @minion-stack/shared@0.1.0` then `npm run build`.)

    6. Verify `minion-shared/dist/index.js` exists and contains the console.warn call.
  </action>
  <verify>

      grep -r "@minion-stack/shared" /home/nikolas/Documents/CODE/AI/minion-shared/src/ && grep '"version": "0.2.0"' /home/nikolas/Documents/CODE/AI/minion-shared/package.json && ls /home/nikolas/Documents/CODE/AI/minion-shared/dist/index.js

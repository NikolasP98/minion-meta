2. Create `packages/lint-config/eslint.config.js` — flat ESLint config (ESM):
```javascript
// @minion/lint-config — flat ESLint config for ESM-based TS/JS projects.
// Consumer usage: in eslint.config.js: `import config from '@minion/lint-config/eslint.config.js'; export default config;`
// Projects that need overrides should extend the array: `export default [...config, { /* overrides */ }];`

import js from '@eslint/js';
import tseslint from 'typescript-eslint';

export default [
  js.configs.recommended,
  ...tseslint.configs.recommended,
  {
    languageOptions: {
      ecmaVersion: 2023,
      sourceType: 'module',
    },
    rules: {
      'no-console': 'off',
      'no-unused-vars': 'off',
      '@typescript-eslint/no-unused-vars': ['warn', { argsIgnorePattern: '^_' }],
      '@typescript-eslint/no-explicit-any': 'error',
      '@typescript-eslint/consistent-type-imports': 'warn',
      'eqeqeq': 'error',
      'no-debugger': 'error',
    },
  },
  {
    ignores: ['dist/', 'build/', '.svelte-kit/', 'coverage/', 'node_modules/'],
  },
];
```

# @minion/lint-config

Three shared lint presets:

| Entry | For | Consumer pattern |
|-------|-----|------------------|
| `oxlint-preset.json` | oxlint-based projects (minion, paperclip-minion) | `.oxlintrc.json`: `{ "extends": ["@minion/lint-config/oxlint-preset.json"] }` |
| `eslint.config.js` | flat-ESLint projects (hub, site, plugins) | `eslint.config.js`: `import config from '@minion/lint-config/eslint.config.js'; export default config;` |
| `prettier.config.js` | all projects | `package.json`: `"prettier": "@minion/lint-config/prettier.config.js"` |

## Peer dependencies

ESLint preset requires these be installed at the consumer:
- `eslint` >= 9
- `@eslint/js`
- `typescript-eslint`

Oxlint preset requires `oxlint` >= 0.15.

Prettier config requires `prettier` >= 3.
